# Curso de Python

Este repositório contém os arquivos do curso de **Programação em Python** que estou realizando como parte do meu aprendizado.  

Ele está organizado da seguinte forma:

- `aulas/` → Contém os códigos e materiais fornecidos durante as aulas.  
- `praticas/` → Contém os exercícios e experimentos que realizei por conta própria para praticar e fixar o conteúdo aprendido.  

## Sobre o curso

Durante o curso, estou aprendendo a:

- Desenvolver programas em Python;  
- Criar algoritmos para resolver problemas.  
- Aplicar conceitos de programação orientada a objetos.  
- Testar e validar os programas.  
- Organizar e documentar códigos de forma clara.  

O repositório será atualizado à medida que realizo novas práticas e exercícios.
